# 🚀 Personal Portfolio Website

A modern, interactive portfolio website built to showcase my projects, technical skills, experience, and achievements as a Software Developer and AI/ML Enthusiast.

Designed with a focus on performance, smooth animations, immersive visuals, and responsive user experience, this portfolio serves as a central hub for recruiters, developers, and collaborators to explore my work.

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/Princeg1204/Portfolio.git
```

### Navigate to project folder

```bash
cd Portfolio
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

This portfolio was created to showcase my technical expertise, projects, and development journey in a visually engaging way.
